# TSSL

Repositorio personal de material de Teoria de Senales y Sistemas Lineales.

## Objetivo

Tener en un solo lugar:

- teoricos y practicos de la cursada actual
- parciales y simulacros
- bibliografia
- tablas de consulta
- material de cursadas anteriores
- notas propias y material generado con Codex

## Estructura

- `01-cursada-actual/`: programa, teoricos, practicos y clases manuscritas
- `02-parciales/`: primer parcial, segundo parcial, simulacros y fotos crudas
- `03-bibliografia/`: libros base y complementarios
- `04-tablas-y-formularios/`: tablas utiles para consulta rapida
- `05-cursadas-anteriores/`: material historico separado por origen
- `06-notas-codex/`: resumentes, notas y resoluciones propias
- `07-archivo-original/`: material preservado por trazabilidad

## Documentacion

- `AGENTS.md`: reglas operativas para agentes
- `docs/indice-material.md`: indice del contenido actual
- `docs/convenciones.md`: criterios de nombres y organizacion
- `docs/observaciones-de-curacion.md`: decisiones y excepciones tomadas
