# Convenciones

## Criterio general

Organizar primero por tipo de material y despues por uso real.
La prioridad no es respetar nombres originales sino mejorar encontrabilidad.

## Nombres

Usar:

- minusculas
- guiones
- palabras semanticas
- prefijos de tipo

Ejemplos:

- `teorico-03-serie-fourier-borgnino.pdf`
- `practico-oppenheim-cap04-transformada-fourier-continua.pdf`
- `parcial-01-2024-cursado.pdf`
- `clase-02-practico-tablet-2026-03-16.pdf`
- `tabla-propiedades-transformada-laplace.jpg`

## Regla para PDFs ambiguos

Si el nombre original dice `capitulo` pero el contenido muestra `practico`, `guia`, `problemas` o `ejercicios`, clasificarlo como practico.

## Regla para duplicados

- Duplicado exacto: conservar una sola copia util.
- Duplicado no exacto pero equivalente: conservar una principal y mover la otra a `07-archivo-original/variantes-duplicadas/`.
- Si una version tiene fecha o contenido mas nuevo, preferir esa.

## Regla para material nuevo

- PDF oficial de catedra: `01-cursada-actual/`
- parcial o simulacro: `02-parciales/`
- libro o apunte largo: `03-bibliografia/`
- tabla de consulta: `04-tablas-y-formularios/`
- material viejo o heredado: `05-cursadas-anteriores/`
- nota hecha en chat: `06-notas-codex/`

## Regla para clases manuscritas

Cada clase debe tener su carpeta:

- `clase-01/`
- `clase-02/`
- `clase-03/`

Dentro de cada una:

- `clase-0N-teorico-tablet-AAAA-MM-DD.pdf`
- `clase-0N-practico-tablet-AAAA-MM-DD.pdf`
